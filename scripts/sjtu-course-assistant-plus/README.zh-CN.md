# 交大选课助手+

[English](README.md) | 简体中文

交大选课助手+ 是用于上海交通大学选课页面的用户脚本。它可以标记课程时间冲突、隐藏冲突教学班、跳转选课社区、显示 jCourse 评分，并在你配置 DeepSeek API Key 后按需生成课程评价总结。

## 功能概览

- 根据已选课程时间，标记当前列表中的冲突教学班。
- 可隐藏与已选课程冲突的教学班或课程面板。
- 为匹配到的课程添加选课社区链接。
- 在可匹配课程旁显示 jCourse 平均评分。
- 配置 DeepSeek API Key 后，可以手动生成课程评价总结。
- 支持自定义总结维度，例如是否点名、是否互动、是否考试等。

## 安装

1. 安装用户脚本管理器，例如 Tampermonkey。
2. 从 Greasy Fork 安装本脚本。
3. 打开上海交通大学选课页面：

```text
https://i.sjtu.edu.cn/xsxk/zzxkyzb_cxZzxkYzbIndex.html
```

脚本只会在匹配的交大选课页面运行。

## 基本使用

页面加载完成后，课程搜索区域附近会出现 `交大选课助手+` 工具栏。

工具栏包含：

- `隐藏冲突课程`：隐藏与已选课程时间冲突的教学班或课程。
- `重新扫描`：在搜索课程、展开课程面板、调整已选课程后重新扫描页面。
- `设置`：配置 API Key、模型、总结维度和隐藏冲突选项。
- `清除错误`：清除脚本显示的错误信息。

脚本会读取页面上的已选课程时间，并与当前展示的教学班时间进行比较。冲突状态会直接显示在课程列表中。

## 选课社区链接与评分

脚本会尝试把选课页面中的课程匹配到 jCourse：

```text
https://course.sjtu.plus
```

匹配成功后，你可能会看到：

- `选课社区` 链接：打开对应的 jCourse 课程页面。
- 课程评分标记：显示 jCourse 平均评分。

jCourse 数据不会在普通扫描时自动请求。只有当你点击选课社区链接或评价总结按钮时，脚本才会请求 jCourse。

## AI 评价总结

脚本当前使用 DeepSeek 生成评价总结。

启用方式：

1. 点击工具栏中的 `设置`。
2. 填入 DeepSeek API Key。
3. 选择模型，或刷新模型列表。
4. 保存设置。
5. 在课程列表中点击评价总结按钮。

脚本不会自动调用 DeepSeek。只有在你点击总结按钮，并且已经配置 DeepSeek API Key 时，才会向 DeepSeek 发送请求。

如果一门课有多个老师，请点击具体教学班旁边的 `本班评价总结`。这类课程通常不能直接生成整门课的总体总结，因为需要先确定老师。

## 设置说明

### DeepSeek API Key

用于生成 AI 评价总结。未配置时，选课社区链接和评分仍可使用，但不能生成 AI 总结。

### DeepSeek Model

默认模型为 `deepseek-v4-flash`。配置 API Key 后，可以在设置面板中刷新可用模型列表。

### jCourse API Key

可选。如果填写，脚本请求 jCourse API 时会通过 Bearer Token 进行认证。

### 总结维度

总结维度用于告诉 AI 从课程评价中提取哪些信息。脚本支持两类维度：

- 是否类：输出“是 / 否 / 未知”，必要时附带简短解释。
- 开放类：输出 20 字以内的短语。

常见维度示例：

- 是否点名
- 是否互动
- 是否考试
- 作业压力
- 给分情况

你可以在设置面板中新增、删除或修改总结维度。维度备注会作为额外要求发送给 AI。

### 隐藏冲突课程

启用后，脚本会隐藏与已选课程时间冲突的教学班或课程面板。你也可以在工具栏中快速切换这个选项。

## 隐私与网络请求

脚本会把设置和缓存保存在用户脚本管理器的本地存储中。

脚本可能请求：

- `course.sjtu.plus`：当你点击选课社区链接或评价总结按钮时。
- `api.deepseek.com`：当你点击总结按钮且已经配置 DeepSeek API Key 时。

脚本在普通页面扫描过程中不会自动把课程信息发送给 DeepSeek。

## 常见问题

如果冲突标记不准确，点击 `重新扫描`。

如果没有出现选课社区链接或评分，可能是课程没有在 jCourse 中匹配到，也可能是课程名、教师或开课院系信息不足。

如果评价总结失败，请检查：

- DeepSeek API Key 是否正确。
- 选择的 DeepSeek 模型是否可用。
- 当前课程是否能匹配到 jCourse。
- 用户脚本管理器是否允许访问脚本声明的外部域名。

如果交大选课系统页面结构更新，脚本可能需要同步更新。
